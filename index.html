Floor Cleaning Bot

Introducing a versatile floor cleaning BOT designed for seamless operation in both automatic and manual modes. Utilizing a Bluetooth module, users can effortlessly control the BOT via a dedicated smartphone app. In automatic mode, an ultrasonic sensor enables obstacle detection, ensuring efficient navigation. Additionally, equipped with a front-mounted mop and water pump, the BOT delivers thorough cleaning with utmost convenience.

Automatic Polarity Reversal Circuit

As part of the Solenix IC Pulse hybrid engine project, which combines both internal combustion and solenoid engines, We focused on the solenoid engine component. We designed and implemented an automatic polarity reversal circuit for the solenoid engine, constructing an H-bridge circuit using solid-state relays controlled by an Arduino Uno microcontroller. This setup efficiently reversed the current direction in the solenoid coils, with solid-state relays handling the high magnitude of current and voltage to ensure reliable operation
